# 🏦 Epic ATM Simulator - C Edition

A professional command-line interface (CLI) application that simulates a real-world Bank ATM experience. This project demonstrates core programming concepts in C, such as dynamic balance management, user input validation, and control flow.

## ✨ Key Features
* **Custom Starting Balance:** Initialize your account with any amount.
* **Secure Deposits:** Add funds instantly with real-time balance updates.
* **Smart Withdrawals:** Built-in logic to prevent withdrawing more than your current balance (Overdraft protection).
* **Interactive Menu:** A clean, loop-based navigation system for a seamless user experience.

## 🛠️ Technical Logic
The program is built using a `do-while` loop to maintain the session active until the user decides to leave, and a `switch-case` structure to handle different banking operations efficiently.
